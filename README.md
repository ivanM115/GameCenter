# Game Center

Proyecto Android que recrea un Game Center con login, menu, puntuaciones y dos juegos (2048 y Tap Frenzy).

## Funcionalidades
- Splash -> login -> menu con perfil y accesos.
- Puntuaciones en SQLite con listado, orden y busqueda.
- Detalle de puntuacion al tocar una tarjeta.
- Juego 2048 con undo y tiempo.
- Segundo juego Tap Frenzy con guardado de score.

## Ejecutar pruebas
```
./gradlew test
```
