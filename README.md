# Game Center

Proyecto Android que recrea un Game Center con login, menu, puntuaciones y juegos.

## Funcionalidades
- Splash -> login -> menu con perfil y accesos.
- Puntuaciones en SQLite con listado, orden y busqueda.
- Detalle de puntuacion al tocar una tarjeta.
- Juego 2048 con undo y tiempo regresivo.
- Juego Puzzle Bobble con guardado de score.
- Juego Arcanoid con guardado de score.
